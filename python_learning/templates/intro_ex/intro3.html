if文（分岐）と条件式

🔵if文
条件に当てはまる場合に処理を行いたい（もし〇〇なら処理△△を実行）時はif文を使用します。
if文の書き方は以下の通りです。
if 条件式:
    条件式が正しい場合に行う処理
Pythonではif文などの処理を書く場合は、その処理の内容にインデント（字下げ）を行う必要があります。
x = 5
if x == 5:
    print("xは5です。")
xは5です。
ここで、「x == 5」という新しい記述の仕方が出てきましたので、次に条件式の書き方を解説したいと思います。
🔵条件式（比較演算子）
「xとyが等しい」を表したい場合は「x == y」のように「=」を２個並べて記述します。
「=」が１個の場合は、右側のデータを左側の変数に代入することを意味するので、条件式の場合は「==」と記述しましょう。
また、「xとyが等しくない」を表したい場合は「x != y」のように「=」の前に「!」を加えて記述します。
他にも、int型のように大小が比べられるものの場合、「x >= y」と記述することで「xがy以上である」という条件式を記述することができます。
「==」や「!=」のように、左辺と右辺を比べる記号を比較演算子と呼びます。比較演算子の一覧を下に示しておきます。 
x == y		#xがyと等しい
x != y		#xがyと異なる
x < y		#xがyよりも小さい
x > y		#xがyよりも大きい
x <= y		#xがy以下である
x >= y		#xがy以上である
x is y		#xがyと等しい
x is not y	#xがyと異なる
x in y		#xがyに含まれる
x not in y	#xがyに含まれない
🔵条件式（ブール演算子）
複数の条件式を組み合わせて新たな条件式を作ることができます。
その際に使用するのが「and」「or」「not」という演算子（ブール演算子）になります。 
条件式A and 条件式B	#条件式Aと条件式Bの両方が正しい(Trueである)
条件式A or 条件式B	#条件式Aと条件式Bの少なくとも１つが正しい(Trueである)
not 条件式A		#条件式Aが正しくない(Falseである)
🔵else、elif
if文の条件に当てはまらなかった場合の処理を書きたい場合は、else文を使うことができます。
y = 12
if y <= 10:
    print("yは10以下です。")
else:
    print("yは10より大きいです。")
yは10より大きいです。
上の例では、if文の条件である「yが10以下」にyが当てはまらなかったため、else文の処理が実行されます。
また、if文の条件が当てはまらなかった場合に別の条件を定義したい場合にはelifを使用します。
(elifは複数個指定することもできます。)
z = 18
if z < 10:
    print("zは小さな数です。")
elif 10 <= z < 100:
    print("zはそこそこの数です。")
else:
    print("zは大きな数です。")
zはそこそこの数です。
